# 接口快速参考

## 管理接口

### 导入Excel数据
```bash
POST /api/import/excel?filePath=D:/path/to/excel.xlsx
```

### 手动触发预警更新
```bash
POST /api/warning/update
```

### 健康检查
```bash
GET /api/health
```

## 查询接口（前端使用）

### 1. 分页查询预警（列表页）
```bash
POST /api/warning/query
Content-Type: application/json

{
  "warningType": "RAIN",      // 可选：RAIN/WIND/VISIBILITY
  "warningLevel": "YELLOW",   // 可选：YELLOW/ORANGE/RED
  "stationCode": "K95",       // 可选：桩号模糊查询
  "stationName": "桂河",      // 可选：路段名称模糊查询
  "startTime": "2026-03-18 00:00:00",  // 可选
  "endTime": "2026-03-18 23:59:59",    // 可选
  "pageNum": 1,               // 页码，默认1
  "pageSize": 20              // 每页大小，默认20
}
```

### 2. 获取统计信息（仪表盘）
```bash
GET /api/warning/statistics

返回：
{
  "totalCount": 150,
  "countByType": {"RAIN": 80, "WIND": 50, "VISIBILITY": 20},
  "countByLevel": {"YELLOW": 100, "ORANGE": 40, "RED": 10},
  "lastUpdateTime": "2026-03-18 15:00:05"
}
```

### 3. 获取最新预警（首页）
```bash
GET /api/warning/latest?limit=10
```

### 4. 查询预警详情
```bash
GET /api/warning/{id}
```

### 5. 按类型查询
```bash
GET /api/warning/type/RAIN        # 暴雨预警
GET /api/warning/type/WIND        # 大风预警
GET /api/warning/type/VISIBILITY  # 能见度预警
```

### 6. 按级别查询
```bash
GET /api/warning/level/YELLOW  # 黄色预警
GET /api/warning/level/ORANGE  # 橙色预警
GET /api/warning/level/RED     # 红色预警
```

## 前端示例

### JavaScript Fetch
```javascript
// 查询预警列表
fetch('http://localhost:8080/api/warning/query', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({pageNum: 1, pageSize: 20})
})
.then(res => res.json())
.then(data => console.log(data.data.records));

// 获取统计
fetch('http://localhost:8080/api/warning/statistics')
.then(res => res.json())
.then(data => console.log(data.data));
```

### Axios
```javascript
// 查询预警
axios.post('/api/warning/query', {
  warningType: 'RAIN',
  pageNum: 1,
  pageSize: 20
}).then(res => {
  console.log(res.data.data.records);
});

// 获取统计
axios.get('/api/warning/statistics')
.then(res => {
  console.log(res.data.data);
});
```

## 数据字典

### 预警类型
- `RAIN` - 暴雨预警

- `YELLOW` - 黄色预警 🟡
- `ORANGE` - 橙色预警 🟠
