**方式3：Maven启动**
```bash
mvn spring-boot:run
```


## 📊 预警规则

### 暴雨预警
| 级别 | 阈值 | 条件 |
|------|------|------|
| 黄色 | 30mm | 实况或预报降水量 > 30mm |
| 橙色 | 50mm | 实况或预报降水量 > 50mm |
| 红色 | 70mm | 实况或预报降水量 > 70mm |

### 大风预警
| 级别 | 阈值 | 条件 |
|------|------|------|
| 黄色 | 10.8 m/s | 实况或预报风速 > 10.8 m/s |
| 橙色 | 13.9 m/s | 实况或预报风速 > 13.9 m/s |
| 红色 | 17.2 m/s | 实况或预报风速 > 17.2 m/s |

### 能见度预警
| 级别 | 阈值 | 条件 |
|------|------|------|
| 黄色 | 1000m | 实况或预报能见度 < 1000m |
| 橙色 | 500m | 实况或预报能见度 < 500m |
| 红色 | 200m | 实况或预报能见度 < 200m |

## 🔄 修改阈值

阈值存储在数据库中，可随时修改：

```sql
-- 修改暴雨黄色预警阈值为35mm
UPDATE warning_threshold
SET threshold_value = 35
WHERE warning_type = 'RAIN' AND level_name = 'YELLOW';

COMMIT;
```

修改后需要重启服务。

## ❓ 常见问题

### Q1: 启动失败，提示数据库连接错误
- 检查虚谷数据库是否启动
- 检查网络是否可访问 `10.159.90.202:36034`
- 检查用户名密码是否正确
- 确认已执行建表脚本

### Q2: Excel导入失败
- 检查文件路径是否正确（使用绝对路径）
- 确认Excel文件存在且格式正确
- 查看日志中的详细错误信息

### Q3: 没有生成预警信息
- 检查是否已导入路段节点数据（`SELECT COUNT(*) FROM geo_line_node`）
- 检查当前气象数据是否达到预警阈值
- 查看日志确认网格数据接口是否调用成功
- 检查网络是否可访问 `http://172.22.96.169`

### Q4: 定时任务没有执行
- 查看日志，确认服务是否正常启动
- 等待6分钟，观察日志输出
- 检查系统时间是否正确

### Q5: 修改阈值后不生效
- 修改数据库后需要重启服务
- 或者等待服务重启时自动加载新阈值
